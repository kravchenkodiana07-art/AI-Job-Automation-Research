
🤖 AI & The Future of Work: Automation Risk Analysis
An interactive research tool exploring how Artificial Intelligence reshapes the global labor market.

This project was developed as part of the ReuseScience initiative by a research team of six. We conducted an in-depth analysis of automation risks across diverse professions, synthesizing data from the World Economic Forum (WEF), McKinsey, and the OECD to provide actionable insights for the modern workforce.

🔍 Research Overview
The core objective was to move beyond the "AI will replace us" narrative and instead provide a nuanced, data-driven look at:

Risk Quantification: Calculating the percentage of task automation for specific roles.

Skill Adaptation: Identifying the "Soft" and "Hard" skills required to remain competitive.

Future Projections: Mapping out the evolution of traditional roles into hybrid human-AI positions.

💻 Technical Implementation
To make our findings accessible, I built a web-based dashboard using Streamlit.

Data Processing: Cleaned and parsed complex CSV datasets containing multi-lingual advice and projections using Pandas.

Dynamic UI: Developed an interface that provides real-time feedback (success/warning/error states) based on the automation risk levels of selected professions.

Logic: Implemented custom formatting to handle nested data structures within the research findings.

🛠️ Tech Stack
Language: Python

Web Framework: Streamlit

Data Analysis: Pandas

Data Source: Curated research from WEF, McKinsey, PwC, and independent surveys.
